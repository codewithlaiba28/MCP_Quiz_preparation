### **1. What is the first step when setting up your FastMCP project directory?**

A) Write your first tool handler function
B) Create and enter a new project directory and initialize the environment
C) Deploy the server to production
D) Write the client initializing handshake

**Correct Option: B**

### **2. In a FastMCP project setup using stateless mode, which configuration flag is often set to make each request independent?**

A) `stateful_http=True`
B) `persistent_session=True`
C) `stateless_http=True`
D) `keep_context=False`

**Correct Option: C**

### **3. When installing dependencies for a FastMCP project, which of the following is a necessary step?**
